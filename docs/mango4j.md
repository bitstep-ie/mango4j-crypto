# mango4j

Code repository: [Repo](TBD)

## Aim
mango4j is a general-purpose Java library that provides reusable utilities and shared crypto tooling used across Bitstep projects.

## Module documentation
- [mango4j-collections](mango4j/mango4j-collections.md) - collection builders, map utilities, reconciliation, and caching.
- [mango4j-utils](mango4j/mango4j-utils.md) - general utilities (conformance, masking, dates, URLs, mapping, threading).
- [mango4j-reflection](mango4j/mango4j-reflection.md) - cached reflection metadata and property accessors.
- [mango4j-validation](mango4j/mango4j-validation.md) - Jakarta validation helpers and reusable constraints.
- [mango4j-hibernate-proxy-resolver](mango4j/mango4j-hibernate-proxy-resolver.md) - Hibernate proxy resolver for mutators.
- [mango4j-crypto-core](mango4j/mango4j-crypto-core.md) - core crypto abstractions and services.
- [mango4j-crypto](mango4j/mango4j-crypto.md) - annotation-driven encryption and HMAC for entities.
- [mango4j-crypto-nonprod-delegates](mango4j/mango4j-crypto-nonprod-delegates.md) - non-production encryption delegates.
- [mango4j-crypto-wrapped-delegate](mango4j/mango4j-crypto-wrapped-delegate.md) - wrapped key encryption delegates.
